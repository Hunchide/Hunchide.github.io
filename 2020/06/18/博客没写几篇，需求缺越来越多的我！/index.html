博客没写几篇，需求却越来越多的我！

发表于 2020-06-18

如题，刺激！

成功发出Hello World的欣喜过后，我就开始有了越来越多的需求！

  1. 嫌弃模板无法满足自己所需功能
  2. 嫌弃文章内容排版难看
  3. 觉得应该有个搜索功能，方便访问者找博客内容（是的，虽然内容篇数都还没翻页！但我已经提前开始膨胀了！）
  4. 最近甚至有了一些在自己本地管理博客内容的需求

对第4点展开讲讲（大概这就是一个用户故事吧）
随着写的博文篇数越来越多，我想在本地更新某篇博文内容的时候，也时常要认真瞅瞅才能找到自己想改的那一篇。
然后我就开始想，本地的source文件夹是不是也有什么更好的管理方式！比如按日期、可搜索、显示tag等等。
